# PSBacktester

Professional Backtester with GUI for trading strategy analysis.

## Features

- MT5 integration for historical data
- CSV data upload support
- Market Mapper with day trading and swing trading modes
- Equity curve visualization
- Comprehensive trading metrics

## Installation

\\\ash
pip install -r requirements.txt
\\\

## Usage

\\\ash
streamlit run dashboard.py
\\\

## Configuration

Set \ENABLE_MT5 = True\ in \mt5_utils.py\ to enable MetaTrader5 features.
